Research Experience

I’m interested in using machine learning to answer neuroscience questions and, in turn, using those insights to build better models of perception and cognition.

EEG Reconstruction from fMRI
Lewis Lab · MIT · 2025–2026 · Cambridge, MA
Advisor: Dr. Laura Lewis
  • Built machine learning models to reconstruct EEG rhythms from simultaneously recorded fMRI data, achieving Pearson correlations of 0.44 (alpha) and 0.41 (delta), improving over prior separate-model results of 0.41 and 0.30.
  • Extending this framework to reconstruct full EEG spectrograms to probe how spatially specific fMRI activity relates to fast neural dynamics.
Categorical Perception in Humans & Models
McDermott Lab · MIT · 2025 · Cambridge, MA
Advisor: Dr. Josh McDermott
  • Designed and ran experiments comparing categorical perception in humans with a biologically plausible speech recognition model; work submitted to the Association for Research in Otolaryngology.
  • Built ML models, generated computational speech stimuli, and conducted human behavioral experiments to align model behavior with perceptual data.

Selected Projects

I enjoy building end-to-end systems that solve real problems, from sports analytics to author-style modeling.

ErgOCR
2025 · Personal Project
Automated transcription of rowing machine workout scores.
  • Developed an application to detect and transcribe rowing machine scores from screen images using object detection, perspective correction, and OCR.
  • Achieved 96% transcription accuracy on a held-out test set, reducing a three-person score-entry process by 10× and deploying it into MIT Crew’s workflow.
LLMimic
2024 · Personal Project
Personalized author-style emulation with fine-tuning and RAG.
  • Built a tool for author style emulation, combining language model fine-tuning with retrieval-augmented generation.
  • Improved style accuracy by 25% using authorship-attribution models and increased stylistic alignment by 85% over baseline using BLEU-based metrics.
  • Designed an evaluation framework integrating BLEU and AA metrics to validate model behavior across diverse writing scenarios.
LuxTronic · ML Intern
2024 · Remote
  • Upgraded object detection pipelines to YOLOv10, yielding ~5% average accuracy improvements on challenging customer datasets while maintaining backward compatibility.
  • Developed a CycleGAN to generate synthetic clear-bottle images from dark bottles, addressing data scarcity for training detection models.
  • Simplified client access to factory data by building a GPT-4o + RAG interface, enabling natural-language queries over operational data.
